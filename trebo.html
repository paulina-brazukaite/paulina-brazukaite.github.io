Trebo - Budget Tracker

Project Description

About

Trebo is an expense tracking and budget planning Web application. It is a personal project. A friend of mine took a part in it by giving the idea and requirements.

Problem

It is hard to track and plan expenses having more than one bank account. There are products in the market that help you to keep all of your accounts in one place. However, they still do have a few disadvantages that burdens budget planning and expense tracking process.

Analogues

- ‘Wallet by Budget Bakers’ web application allows to upload different banks’ statements in one place.

- ‘Revolut’ has a great expense categorising feature that is as an example for this project..

Pains

- ‘Wallet’ does not allow to create a new category for existing transaction. User has to choose from the existing ones.

- 'Revolut’ is not capable of sorting categories based on the date. User can choose just a weekly or monthly Analytics showcase.

Requirements

- To import bank reports using ‘.csv’ and ‘.xls’ formats.

- The dashboard has to include: date picker; ability to edit the category of the record; income/outcome and category filter; ability to add transaction manually.

- Categories summary needs to show the percentage marking that indicates trending categories in spending.

- There has to be a representation of a current cashflow including income and outcome. It also needs to have a date picker.

Main Goal

- To design an easy to use platform that would help to track and plan expenses having all records from different banks in one place.
